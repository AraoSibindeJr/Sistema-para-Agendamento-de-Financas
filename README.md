# FinAgenda - Sistema de Agendamento para Pagamentos de Finanças

## Descrição do Projeto

O FinAgenda é uma aplicação web desenvolvida no âmbito da cadeira de Programação Web 1, do 3º ano, 1º semestre do curso de Engenharia e Ciência dos Computadores. O objetivo principal do projeto é criar um sistema de agendamento online que permita aos utilizadores marcar datas e horários para realizar pagamentos relacionados com serviços de finanças.

A plataforma visa simplificar a gestão de compromissos financeiros, oferecendo uma interface intuitiva para os cidadãos e uma área de gestão para a administração.
